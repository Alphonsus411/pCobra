# Archivo de artefactos retirados

Este directorio conserva material histórico retirado del árbol principal del
producto.

## Política

- No forma parte de la documentación pública principal.
- No debe enlazarse desde índices, guías de producto o tablas canónicas.
- No está cubierto por la suite normal de regresión ni por las validaciones de
  política del árbol principal.
- Su contenido se conserva únicamente como referencia histórica y de migración
  interna.
