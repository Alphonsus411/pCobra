# Minilenguaje

## Introducción
En esta entrada se presenta un vistazo rápido al minilenguaje de Cobra.

## Ejemplo rápido
```cobra
imprimir("Hola mundo")
```
Para transpilar este código a Python ejecuta:
```bash
cobra archivo.co --lenguaje python
```
