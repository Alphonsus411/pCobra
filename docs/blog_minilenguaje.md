# Minilenguaje

## Introducción
En esta entrada se presenta un vistazo rápido al minilenguaje de Cobra.

## Ejemplo rápido
```cobra
imprimir("Hola mundo")
```
Para ejecutar y construir con la CLI unificada:
```bash
cobra run archivo.co
cobra build archivo.co
```
